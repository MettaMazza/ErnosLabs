Ern-OS

A small operating system written entirely in plain-English Ernos — kernel, shell, filesystem, desktop, all of it. Self-contained, works with no network, and self-hosting: it rebuilds itself from a C compiler alone. Every part of it, including the system itself, is written in language you can read.

You talk to Ern-OS in sentences — make a folder called letters, write a note called hello saying good morning, rebuild the system.
